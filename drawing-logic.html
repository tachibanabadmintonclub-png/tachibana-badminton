🏸 抽選ロジック解説

「抽選する」ボタンが何をしているのかを詳しく説明します

デフォルト設定での抽選フロー

デフォルト設定：
• スキルレベルマッチング：有効 (指定レベルのメンバーを優先)
• 待機優先：有効 (長く待っている人を優先)
• 連続ゲーム制限：なし
1
本日の参加メンバーから「利用可能な人」をピックアップ
除外される人：
  • すでに別のコートに配置されている人
  • ステータスが「休憩中」または「限界」の人
  • 連続ゲーム制限に引っかかっている人
例：本日参加12名 → 利用可能10名
2
コート1、コート2の順で抽選
各コートに設定されたレベルに応じて配置戦略を変更します：
コート1（レベル1指定）: レベル1のメンバーを優先
コート2（全員対象）: 全員の中からバランスよく選出
3
優先度にしたがって4人を選出
待機優先の場合： より長く待っている人を優先
試合数優先の場合： 試合数が少ない人を優先
待機時間：A(20分) > B(10分) > C(5分) > D(2分)
→ 配置順：A, B, C, D
4
コートに4人を配置
4人揃わない場合は、空きスロットはマニュアルで埋める必要があります。
例：利用可能2人のみ → 2人配置 + 2スロット空き

設定を変更するとどうなるか？

1️⃣ 待機優先 → 試合数優先に変更

設定変更： 「待機優先」をOFF

抽選ロジック： 待機時間ではなく、試合数が少ない人を優先
設定 優先される人 効果
デフォルト
待機優先		 長く待っている人 新参者や試合が少ない人は待っている間に機会が来ない可能性
変更後
試合数優先		 試合数が少ない人 全員が均等に試合機会を得やすくなる

2️⃣ 連続ゲーム制限を有効にする

設定変更： 「連続ゲーム制限」をON、最大2ゲームに設定

抽選ロジック： 連続で3ゲーム目をしようとする人を自動で除外
制限なし場合：
Xさんが1日に10ゲーム連続でプレイ可能 → 疲労が溜まる
制限あり場合（最大2ゲーム）：
Xさんが2ゲーム目まで → 3ゲーム目で自動除外 → 休憩 → 回復後に再度抽選可能

3️⃣ コート別レベル設定で「スキルマッチング」をカスタマイズ

設定例：
• コート1 = Lv2（中級者向け）
• コート2 = 全員対象（初心者〜上級者混合）

効果： Lv2の人が集中してプレイできる環境を実現
例：中級者大会モード
コート1 = Lv2（本格的なラリー）
コート2 = Lv2（本格的なラリー）
コート3 = ALL（初心者向け練習）
→ Lv2が多い日は、Lv2同士で実力が拮抗した試合ができる

よくある質問

Q1: 同じ人ばかり選ばれるのはなぜ？

A: 待機優先の場合、待機時間が長い人が繰り返し選ばれます。

解決方法：
  • 設定から「試合数優先」に変更して、試合経験の少ない人を優先する
  • または、連続ゲーム制限を設定して、同じ人が連続プレイするのを制限する

Q2: 「メンバー不足」と表示されたら？

A: 利用可能なメンバーが4人未満の場合、空きスロットが発生します。

対応方法：
  • 画面上の空きスロット（グレー）をタップして、メンバーを手動で配置する
  • または、本日タブで参加メンバーを追加する

Q3: レベル1とレベル2が混在するコートでうまく配置できない

A: コート別にレベルを指定しているため、指定レベルが少ない場合は補充されます。

例： コート1をLv1指定で、本日Lv1が1人のみ → Lv1を1人配置 + Lv2から3人補充

調整方法：
  • そのコートを「全員対象」に変更してレベル混合を容認する
  • または、参加メンバーのレベル構成を調整する

Q4: 「連続ゲーム制限」の「インターバル」とは？

A: 前の試合終了から次の試合開始までの最小待機時間です。

例： インターバル30分、Aさんが15:00に試合終了
→ 15:30までは次の抽選対象から除外
